# axiom-sqlalchemy

Axiom SQLAlchemy integration — models, mixins, repositories, DAO

## Installation

```bash
uv add axiom-sqlalchemy
```

## Usage

```python
from axiom.sqlalchemy import ...
```

## Planned Components

- `models` — base SQLAlchemy declarative models
- `mixins` — reusable model mixins (timestamps, soft delete, etc.)
- `repositories` — generic repository pattern implementation
- `dao` — data access objects layer
